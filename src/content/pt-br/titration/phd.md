A tese de doutoramento, intitulada "Fungi in Cerrado plants: an integrative approach", resultou no desenvolvimento de duas ferramentas bioinformáticas: "GeneConnector" e "Classeq". A primeira foi desenvolvida para consolidar informações genéticas a partir dos metadados intrínsecos aos registros do GenBank. A segunda, "Classeq", é um classificador de sequências biológicas que emprega filogenias pré-construídas para posicionar sequências de DNA com alto desempenho, baixo custo computacional e reduzida taxa de erro, por meio de um ambiente acessível via API projetado para integração com máquinas.
