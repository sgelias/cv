O foco profissional atual está concentrado no desenvolvimento de ferramentas bioinformáticas para o setor de Agrobiotecnologia, abrangendo soluções para análise de dados genômicos e metagenômicos, bem como o processamento de sequências biológicas. Esforços paralelos têm sido direcionados ao design e entrega de interfaces web-nativas de alta disponibilidade que suportam soluções de software para o agronegócio brasileiro.
